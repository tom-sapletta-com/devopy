# Skrypty pomocnicze

Ten katalog zawiera różne skrypty pomocnicze używane w projekcie Devopy, które nie pasują do innych kategorii.

## Przeznaczenie

Skrypty w tym katalogu służą do:

1. Automatyzacji zadań związanych z rozwojem projektu
2. Wsparcia procesów CI/CD
3. Generowania dokumentacji
4. Innych zadań pomocniczych

## Dodawanie nowych skryptów

Przy dodawaniu nowych skryptów do tego katalogu należy:

1. Nadać im uprawnienia wykonywania (`chmod +x nazwa_skryptu.sh`)
2. Dodać krótki opis ich funkcji w tym pliku README
3. Zapewnić, że skrypty zawierają odpowiednie komentarze wyjaśniające ich działanie
